The browser DOM section.

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

It is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. That way, programming languages can connect to the page.

Debugging with DOM is the part where the browser rebuilds missing parts of code and you see the changes in the DOM section of the browser. Chrome > right click > Inspect > Elements > DOM Breakpoints
